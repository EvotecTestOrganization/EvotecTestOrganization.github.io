## Output from Technet Gallery

This is a little script that gathers only essential information on network printers.  The specific fields are:

1. SystemName (Server)

2. Name (Printer Name)

3. Location

4. Portname (IP Address)

5. DriverName

6. Capabilities

7. Comment
